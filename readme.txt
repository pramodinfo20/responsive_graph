Folder contains 2 datset files (test_dataset_1.hdf5, test_dataset_2.hdf5) and 2 python scripts (test_dataset_1.py, test_dataset_2.py).


Instruction to Run the code

1) open command prompt or terminal
2) Make sure you are in the same directory as the folder
3) run the python script as below
	python test_dataset_1.py
	python test_dataset_2.py


The output of the python script will be an interactive graph wherein the first graph shows the visualization of glucose over time
the second graph shows the corresponding measurement when you click on any point in the first graph. (screenshots shared)
